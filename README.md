# Kodai Utsunomiya Website

This repository hosts the public website for Kodai Utsunomiya.

## Developer Notes

### Stack
- Astro

### Local Development
```sh
npm install
npm run dev
```

### Build
```sh
npm run build
npm run preview
```

### Content
- Main page: `src/pages/index.astro`
- Blog posts: `src/content/blog/*.md`
- Layout: `src/layouts/BaseLayout.astro`
- Assets: `public/`
