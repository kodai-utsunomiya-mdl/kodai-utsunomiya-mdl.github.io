[Bachelor Thesis] 中間層のニューロン数が層によって異なるオーダーを持つ深層ニューラルネットワークの学習ダイナミクスの解析

GitHub Repository / Paper (PDF)

概要: 層ごとに幅のオーダーが異なる (ボトルネック構造などを持つ) 深層ニューラルネットワークにおいて，学習が安定する条件を導出． 信号伝播のオーダー評価を行い，パラメータの更新量がネットワーク内の最小の層幅に制約されるという規則を特定．これに基づき，初期化と学習率等を動的に調整する手法 Dynamic Parametrization を提案．
